## Features

- Accepts an input array from the user.
- Performs **Bubble Sort** and **Selection Sort** simultaneously for comparison.
- Displays all sorting passes for both algorithms.
- Shows the total number of **key comparisons** performed in each pass.
- Shows the total number of **swaps** performed in each pass.
- Determines and indicates the **Best Case**, **Average Case**, or **Worst Case** based on the number of swaps performed.
- Highlights elements that reach their final sorted position in each pass. *(Currently not implemented.)*
